# MapGPS-基于GPS的实时位置标记系统

## 统整体设计如下
1，从串口读取GPS数据
2、校验数据
3、解析数据
4、连接mysql
5，数据入库
6，PHP连接mysql的
7，从数据读取数据
8，查询显示地图
